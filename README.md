# Secure AI Gateway

企業での生成AI利用における「情報漏洩」を防ぐための、セキュリティゲートウェイ・システムです。
Go言語で開発したプロキシサーバーが、AIへのリクエストを常時監視し、機密情報の流出を阻止します。

## 機能
- **キーワード検閲**: 「社外秘」などの禁止用語をブロック
- **DLP (情報漏洩対策)**: 正規表現によるメールアドレス・電話番号の自動検知
- **シークレットスキャン**: ソースコード内のAPIキー送信を防止
- **管理ダッシュボード**: リアルタイムでブロック状況を可視化・分析 (HTML/Chart.js)
- **社内チャットUI**: 社員が利用するためのWebチャット画面

## 使用技術
- **Backend**: Go (net/http)
- **Frontend**: HTML5, CSS3, JavaScript
- **Library**: Chart.js (グラフ表示)

## 使い方
1. リポジトリをクローン
2. `go run main.go` で起動
3. `http://localhost:8080` にアクセスしてチャットを利用
4. `http://localhost:8080/admin` で監視ログを確認
